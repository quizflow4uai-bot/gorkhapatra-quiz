About

This quiz application automatically generates questions from the Gorkhapatra Online Loksewa preparation special article. Questions are converted into multiple-choice format via AI (Groq LLaMA 3.3 70B).

This application is hosted on GitHub Pages and publicly available.

Features

  • Daily updated quizzes
  • Multiple choice questions
  • Timed quiz sessions
  • Progress tracking
  • Local result calculation

Tech Stack

Scraper: Python + BeautifulSoup
AI Generator: Groq API + LLaMA 3.3 70B
Frontend: Vanilla JS + CSS
Hosting: GitHub Pages

Source Code

This project is open source and available on GitHub.
