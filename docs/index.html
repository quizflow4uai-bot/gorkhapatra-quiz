About

From समसामयिक

This quiz application automatically generates questions from the Gorkhapatra Online Loksewa preparation special article. Questions are converted into multiple-choice format via AI (Groq LLaMA 3.3 70B).

Features

  • Multiple choice questions
  • Timed quiz sessions
  • Progress tracking
  • Local result calculation

Tech Stack

Frontend: Vanilla JS + CSS
Hosting: GitHub Pages
